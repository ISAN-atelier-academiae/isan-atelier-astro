# 🏛️ Portal Principal de ISAN atelier academiae C.A.

Bienvenido a la consola soberana de edición. Puedes escribir o modificar cualquier bloque de texto en este archivo Markdown (`.mdx`) directamente desde tu bóveda de Obsidian y el contenido se renderizará automáticamente en la planta baja de tu portal principal al presionar el botón **🛡️ Publicar Campus Soberano**.

### ⚙️ Instrucciones de Personalización:

1. **Editar Cabecera:**
   * Cambia el valor de `title` en el bloque superior para actualizar el título SEO de la web y el título de neón del explorador.
   * Cambia `description` para alterar el resumen de búsqueda de Google.
2. **Estructura del Portal:**
   * Puedes cambiar el orden o eliminar los componentes superiores importados (como `<MediathequeFilters />` o `<NewsletterSubscription />`) simplemente borrándolos o comentándolos en este archivo.
3. **Contenido Adicional:**
   * Cualquier texto, imagen o sección que escribas aquí abajo (en formato Markdown estándar) aparecerá de forma inmediata en la página de inicio, debajo de la consola táctil.
